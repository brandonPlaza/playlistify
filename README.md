<h2>Welcome to Playlistify</h2>
<p>Playlistify aims to make it easier to start playlists or to show you to new music through sudo random playlist generation. Currently it is a work in progress but the proposed features include:</p>
<ul>
  <li>Generating playlists based on provided moods.</li>
  <li>Editing generated playlists.</li>
  <li>Analyzing existing playlists from a user and creating a playlist with songs from the same artists.</li>
  <li>Adding new music to existing playlists based off of provided moods.</li>
</ul>
<p>Playlistify is hosted via netlify and can be found at <a href="https://plaza-playlistify.netlify.app/">this url</a></p>
